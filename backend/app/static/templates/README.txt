Este é um arquivo de placeholder para o template do Power BI.

Para utilizar esta funcionalidade:

1. Crie um relatório no Power BI Desktop
2. Conecte-se aos dados da planilha Excel gerada pelo sistema
3. Desenvolva os visuais necessários para análise de frotas
4. Salve como template (.pbit)
5. Substitua este arquivo pelo seu template Power BI

Quando estiver pronto, coloque o arquivo .pbit neste diretório com o nome "frotas_template.pbit"
